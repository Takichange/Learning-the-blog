.gitignore文件不生效解决方法
使用git提交代码希望忽略某些文件时，需要配置.gitignore。

不过已经提交后文件，再修改.gitignore文件会没有效果。

用下面命令解决：

git rm -r --cached .
 
git add .
 
git commit -m 'update .gitignore'
